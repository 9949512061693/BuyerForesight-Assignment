# 📊 User Directory Dashboard

A responsive and user-friendly User Directory Dashboard built using React.js and Vite. The application fetches user data from a REST API and allows users to search, sort, and explore detailed user profiles with a clean and modern UI.

---

## 🚀 Features

- 🔍 Search users by name or email
- 🔃 Sort users (Ascending / Descending)
- 📄 Detailed user profile view
- ⚡ Fast performance with Vite
- 🔁 API integration with loading & error handling
- 🎯 Responsive design for all screen sizes

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- React Router DOM
- Bootstrap & Custom CSS
- REST API (JSONPlaceholder)
